# Aura AI - Curso de Inteligencia Artificial

Bienvenido a **Aura AI**, una plataforma educativa interactiva diseñada para enseñar conceptos avanzados de Inteligencia Artificial de forma práctica y visual.

## 🚀 Características Principales

- **Dashboard de Estudiantes:** Un entorno de aprendizaje personalizado donde los alumnos pueden acceder al material del curso.
- **Módulos Interactivos:**
  - **Regresión Lineal:** Explicaciones teóricas, ejemplos de casos de uso, simulador interactivo y herramienta de análisis de datos mediante archivos CSV.
  - **Algoritmos Genéticos:** Fundamentos evolutivos, laboratorio de evolución (simulador) y centro de evaluación.
  - **Redes Neuronales:** Arquitectura profunda, visualizador de flujo de información y exámenes de certificación.
- **Sistema de Evaluación:** Cuestionarios interactivos y exámenes de certificación para validar el aprendizaje.
- **Panel Administrativo:** Herramienta integral para gestionar solicitudes de registro, aprobar accesos y administrar a los estudiantes inscritos.

## 🛠️ Tecnologías Utilizadas

- **Frontend:** HTML5, CSS3 (con variables CSS y diseño Glassmorphism), Vanilla JavaScript.
- **Backend/API:** PHP y Node.js (con Express).
- **Base de Datos:** SQL.
- **Diseño UI/UX:** Interfaz moderna y dinámica con animaciones fluidas, modo oscuro y tipografía Inter.

## 📥 Instalación y Uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Pedroc9/pedro-cueto-laboratorio-.git
   ```
2. Configura tu servidor local (como XAMPP, AppServ o Node.js) para ejecutar el backend.
3. Importa el archivo `database.sql` a tu gestor de base de datos MySQL.
4. Abre `index.html` en tu navegador o accede a través de tu entorno de desarrollo local.

## 🔐 Panel de Administración

Para acceder al panel administrativo y gestionar a los estudiantes:
1. Haz clic en el botón "Panel Admin" en la esquina superior izquierda.
2. Ingresa la contraseña de administrador configurada en el sistema.
