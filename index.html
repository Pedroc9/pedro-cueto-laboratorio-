Regresión Lineal

¿Qué es la Regresión Lineal?

La regresión lineal es un algoritmo de aprendizaje supervisado que se utiliza para predecir un valor numérico continuo basado en la relación entre variables.

Concepto Clave

Buscamos la línea que mejor se ajuste a los datos minimizando el error cuadrático medio.

Casos de Uso Reales

  • Predicción de precios de viviendas.
  • Estimación de ventas mensuales.
  • Análisis de tendencias económicas.

Redes Neuronales

¿Qué es una Red Neuronal?

Las redes neuronales artificiales son modelos inspirados en la estructura biológica del cerebro humano para procesar información y aprender de patrones complejos.

Componentes Básicos

  • Neuronas: Unidades de procesamiento.
  • Capas: Entrada, Ocultas y Salida.
  • Pesos y Sesgos: Parámetros que la red ajusta para aprender.
  • Funciones de Activación: (ReLU, Sigmoide) que deciden si la neurona se "activa".
